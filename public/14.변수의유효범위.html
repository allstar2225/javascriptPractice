- 지역 변수(local variable) -

+ 함수 내에 선언된 변수. 변수가 선언된 내에서만 사용가능. 함수가 종료되면 메모리에서 삭제됨.

- 지역 변수(local variable) -

+ 함수 내에 선언된 변수. 변수가 선언된 내에서만 사용가능. 함수가 종료되면 메모리에서 삭제됨.

- 전역 변수1(global variable) -

+ 함수 외부에서 선언된 변수. 어느 영역에서나 접근 가능. 웹페이지가 닫혀야 메모리 삭제.

- 전역 변수2(global variable) -

+ 함수 내부에서 var없이 선언시, 전역 변수로 사용 가능하다

- 전역 변수3(global variable) -

+ 전역변수와 같은 이름의 지역 변수를 선언하면, 함수 내에선 지역 변수만 사용한다.

- 전역 변수4(global variable) -

+ 해당 블록에 전역 변수를 호출 하려면, 해당 객체가 window 객체의 property임을 명시하면 된다.
