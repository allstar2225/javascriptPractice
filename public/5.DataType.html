- 숫자타입 (Number Type) -

+ 숫자타입의 데이터를 출력

- 문자타입 (String Type) -

+ 문자타입의 데이터를 출력

- 불리언타입 (Boolean Type) -

+ 데이터의 참과 거짓을 출력

- 심볼 타입 (Symbol Type) -

+ object의 property를 위한 식별자로 사용

- typeof 타입 (typeof Type) -

+ 피연산자의 타입을 반환하는 피연산자가 단 하나뿐인 연산자

- null과 undefined 타입 (null, undefined Type) -

+ 아직 값이 정해지지 않은 것은 null, 타입이 정해지지 않은것은 undefined

- 객체 타입 (object Type) -

+ 기본 타입인 객체. property 나 method 같은 이름으로 묶어 놓는다.
