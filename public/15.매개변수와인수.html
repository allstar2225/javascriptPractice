- 매개변수(parameter) 와 인수(argument) -

+ 매개변수는 전달받은 인수를 함수 내부로 전달하기 위해 사용하는 변수

+ 인수는 함수가 호출될때 함수로 값을 전달해주는 값.

- if를 사용한 응용법 -

+ if를 사용해 조건에 만족하지 않아도 값을 띄워줄수 있다.

- 인수(argument) 객체 -

+ 함수의 정의보다 많은 인수가 전달되면 대입되지 않는다. 대신 인수 객체를 사용하면 각각의 인수에도 접근 가능.

- 디폴트 매개변수 (default parameter) -

+ ECMAScript6부터 새롭게 정의되었으며, 함수를 호출할 때 명시된 인수를 전달 못받을 경우 사용할 기본값 (원래 기본값은 undefined)

- 나머지 매개변수(rest parameter) -

+ ECMAScript6부터 새롭게 정의되었으며, 생략 접두사(...)를 사용, 특정 위치 인수부터 마지막 인수까지 한번에 지정 가능
