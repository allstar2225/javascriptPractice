- 자바스크립트의 객체(object) -

+ 그냥 간단하게 사물이라 생각하면 된다. javascript에서의 기본 타입은 객체로써 각각의 property값을 가진다.

- 객체의 프로퍼티 -

+ 객체에 property를 참조하는 방법

- 객체의 생성방법 (리터럴 표기 (literal notation))-

+ 객체를 생성하는 가장 쉬운방법은 리터럴 표기이다.

- 객체의 생성방법 (생성자 함수 (constructor function))-

+ new 연산자를 사용하여 객체 생성을 초기화 한다. 사용되는 메소드는 생성자이며, 새롭게 생성되는 객체를 초기화 한다.

- 객체의 생성방법 (Object.create())-

+ 지정된 프로토타입 객체와 프로퍼티를 가진 새로운 객체를 만들어준다. 이 메소드를 이용하면 프로토타입 객체를 직접 명시할수 있어서 유용하다.

+ Object.create(프로토타입객체, { 객체명 : {프로퍼티값}, 객체명 : {프로퍼티값}, ... }) ,

- 프로토타입(prototype) -

+ 객체 생성자 함수를 작성하여 프로토타입을 생성. 가장 기본적이다.

+ 추가로 this키워드는 해당 키워드가 사용된 자바스크립트 코드 영역을 포함하고 있는 객체를 가리킨다. 예를 들어, 메소드 내부에서 사용된 this는 해당 메소드를 포함하고 있는 객체를 가리키고, 객체 내부에서 사용된 this는 객체 그 자신을 가리킨다. 이러한 this는 변수가 아닌 키워드 이므로, 사용자가 임의로 가리키는 값을 바꿀수 없다.

- object에 property 및 method 추가 -

+ 이미 생성된 객체에 새로운 프로퍼티나 메소드를 추가하는 방법

- 프로토타입에 프로퍼티 및 메소드 추가. -

+ 생성자 함수에 직접 추가해야만 이후에 생성되는 모든 다른 객체에 적용.

- 프로토타입 프로퍼티 (prototype property) -

+ 현재 존재하고 있는 prototype에 새로운 property나 method를 손쉽게 추가한다.

- 객체 프로퍼티 삭제 -

+ delete 키워드를 사용하여 객체 프로퍼티를 삭제 할수있다. 단, 함수나 변수에 사용하면 동작하지 않는다.

- 객체의 순회 -

+ for/in 문을 사용하여 프로퍼티를 순회한다.
이외에도 Object.keys()로 해당 객체가 가진 고유 프로퍼티 중에서 열거할 수 있는 프로퍼티 이름을 배열에 담아 반환한다.
Object.getOwnPropertyNames()는 해당 객체가 가진 모든 고유 프로퍼티의 이름을 배열에 담아 반환

- 객체간 비교 -

+ 같은 프로퍼티값을 가져도, 프로토타입으로 생성된 변수가 가리키는 객체 레퍼런스 그 자체를 저장하는것이 아닌 객체가 위치한 주소를 저장하는것이기 때문에 같지 않다.
