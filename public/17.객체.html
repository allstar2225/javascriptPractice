- 자바스크립트의 객체(object) -

+ 그냥 간단하게 사물이라 생각하면 된다. javascript에서의 기본 타입은 객체로써 각각의 property값을 가진다.

- 객체의 프로퍼티 -

+ 객체에 property를 참조하는 방법

- 객체의 생성방법 (리터럴 표기 (literal notation))-

+ 객체를 생성하는 가장 쉬운방법은 리터럴 표기이다.

- 객체의 생성방법 (생성자 함수 (constructor function))-

+ new 연산자를 사용하여 객체 생성을 초기화 한다. 사용되는 메소드는 생성자이며, 새롭게 생성되는 객체를 초기화 한다.

- 객체의 생성방법 (Object.create()))-

+ 지정된 프로토타입 객체와 프로퍼티를 가진 새로운 객체를 만들어준다. 이 메소드를 이용하면 프로토타입 객체를 직접 명시할수 있어서 유용하다.

+ Object.create(프로토타입객체, { 객체명 : {프로퍼티값}, 객체명 : {프로퍼티값}, ... }) ,
