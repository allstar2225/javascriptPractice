- window.alert() -

+ 새창을 띄워 사용자에게 데이터 전달

- innerHTML 프로퍼티 -

+ document의 getElementByID() 나 getElementsByTagName()로 HTML요소 선택. innerHTML을 사용하면, html 요소의 내용(content)나 속성(attribute)값을 손쉽게 바꿀수 있다.

이 문장이 바뀔것이다.

- document.write() -

+ 웹페이지가 로딩될때 실행되며, 가장먼저 데이터를 출력. 보통 테스트나 디버깅을 위해 쓴다.

- console.log() -

+ 웹 브라우저의 콘솔을 통해 데이터 출력. 디버깅 할때 굉장한 도움이 된다.

크롬의 f12를 눌러서 console칸에 가면 데이터를 확인 할수 있다.
