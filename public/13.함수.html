- 기본 함수 -

+ 특별 목적의 작업을 수행하도록 독립적인 블록. 필요할때마다 호출하여 반복적으로 작업 수행 가능.

+ function 함수명 (매개변수1, 매개변수2,.....) { 호출 되었을때 실행하고자 하는 실행문 } 

        function addNum(x, y) {
            return x + y;
        }
        document.write(addNum(2, 3));


- 반환문 -

+ 함수의 실행결과값을 외부로 보내고자 할때 사용. 함수의 실행을 중단하고, 표현식의 값을 호출자에게 반환.



- 변수에 함수를 저장 -

+ 함수는 값이 될수도 있기에 변수에 대입하여 사용할수도 있다.

        function sqr(x){
            return x; // 호출자에게 x의 값을 반환한다.
        }
        var sqrNum = sqr; // sqrNum 이라는 변수에 sqr함수를 담아둔다.
        document.write(sqr(4); //함수 sqr을 호출하여 직접 값을 출력
        document.write(sqrNum(4)); // 변수를 함수처럼 사용
