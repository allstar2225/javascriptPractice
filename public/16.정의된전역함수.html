- eval() -

+ 문자열로 표현된 javascript 코드를 실행

- isFinite() -

+ 전달된 인수값이 유한한 수인지 검사하여 결과를 반환. 숫자가 아니면 숫자로 반환하여 검사.

- isNaN() -

+ 전달된 인수값이 NaN인지 검사. 결과를 반환. 숫자가 아니라면 강제로 변환하여 검사.

- parseFloat() -

+ 문자열을 파싱하여 부동 소수점의 수로 반환

- parseInt() -

+ 문자열을 파싱하여 정수로 반환

- encodeUI(), encodeURIComponent() -

+ encodeUI()는URI에서 주소를 표시하는 특수문자를 제외하고, 모든 문자를 이스케이프 시퀀스 처리하여 부호화함. 하지만 encodeURIComponent()는 부호화 하지 않은 모든 문자까지 이스케이프 시퀀스 처리함

- dencodeUI(), dencodeURIComponent() -

+ dencodeUI()는 다른 방법으로 만들어진 URI를 해독. dencodeURIComponent()는 encodeURIComponent()나 다른 방법으로 만들어진 uri컴포넌트를 해독

- escape() , unescape() -

+ 전달받은 문자열에서 특정 문자들을 16진법 이스케이프 시퀀스 문자로 변환. unescape() 전달받은 문자열에서 escape() 함수나 다른 방법으로 만들어진 16진법 이스케이프 시퀀스 문자를 원래의 문자로 전환

- number() -

+ 전달받은 객체의 값을 숫자로 반환

- String() -

+ 전달받은 객체의 값을 문자열로 반환
