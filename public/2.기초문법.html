- 기본문법 -

+ 실행문은 세이콜론(;) 으로 구분

        - 위의 문법 -
        var x = 10;
        document.write(x);

- 기본문법2 -

+ 대소문자를 구분하기때문에 변수나 함수의 이름, 예약어 작성시 대소문을 확실히 구분해야한다.

        - 위의 문법 -
        var Java = 10;
        var java = 20;
        document.write(Java + " 이 값은 Java ");
        document.write(java + " 이 값은 java ");

- 리터럴(literal) -

+ 직접 표현되는 값

        - 위의 문법 -
        document.write(12); // 숫자 리터럴
        document.write("안녕"); // 문자 리터럴
        document.write(true); // 불리언 리터럴

- 식별자(identifier) -

+ 변수나 함수의 이름을 작설할떄 사용하는 이름

        - 위의 문법 -
        var firstJava = 20; // camel Case 방식
        var first_java = 30; // underscore Case 방식
        document.write(firstJava);
        document.write(first_java);
